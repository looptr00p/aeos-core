# Review Cadence

## Purpose

Define a lightweight operational review rhythm for AEOS execution readiness.

## Daily

- task continuity review
- unresolved blockers
- workflow integrity

## Weekly

- governance review
- workflow friction review
- incident review
- context continuity review

## Monthly

- operational metrics review
- governance drift review
- workflow effectiveness review
- escalation analysis

## Quarterly

- architecture review
- governance simplification review
- organizational scaling review
